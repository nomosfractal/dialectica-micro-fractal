# Libro en Markdown (Español)

Esta carpeta contiene la versión en **español** de *Dialéctica Micro-Fractal* en formato Markdown.  
Es la versión accesible, editable y navegable del libro, diseñada para facilitar:

- la lectura directa en GitHub,  
- la colaboración mediante *pull requests*,  
- la creación de nuevos capítulos o extensiones,  
- correcciones conceptuales y editoriales.

Cada archivo representa un capítulo independiente, lo que permite editar, sustituir o reorganizar el contenido sin afectar otras partes del texto.

Las actualizaciones realizadas aquí se integran posteriormente en las versiones PDF y LaTeX del proyecto.
